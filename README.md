Official repository for the [AnAi protocol landing page](#).

## Getting started

### Install dependencies with yarn

```ssh
yarn
```

### Start the development server

```ssh
yarn start
```

## Useful commands

Check Typescript code

```ssh
yarn tsc
```

Lint code

```ssh
yarn lint
```

Format code

```ssh
yarn format
```

Generate production build

```ssh
yarn build
```

Start application with production build

```ssh
yarn serve
```
